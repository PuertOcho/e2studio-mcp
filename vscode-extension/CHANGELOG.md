# Changelog

## 1.0.0 — 2026-03-19

Stable release.

- Sidebar panel for project, debugger and build configuration selection
- Build, clean, rebuild and flash flows integrated with e2 Studio toolchains
- Hardware debug sessions for Renesas RX with launch-file-aware configuration
- ADM virtual console integration
- Project folder selection with improved refresh and progress behavior
- Probe status handling adjusted for normal E2 Lite disconnect/reconnect behavior during debug
- Debug startup now reuses an existing built binary when available
- ADM Virtual Console output channel
- MCP server integration (Model Context Protocol)
- Auto-detection of Renesas toolchain paths
- Support for E2 Lite, E1 and J-Link probes
